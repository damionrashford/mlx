performance. An AUC of zero represents
a very bad classifier, and an AUC of one will represent
an optimal classifier. Like any other
evaluation metric, AUC has advantages
and disadvantages. It's advantages
include the fact that AUC gives a single number, which makes it easy to
compare classifiers. Another advantage is
that AUC does not require specifying a
particular decision threshold. Instead, you can think
of AUC as a summary of the performance of
a classifier over all possible
decision thresholds. On the other hand, like any
other single number metric, AUC loses information about classifier trade-offs and
the shape of the ROC curve. It might be that two
classifiers have similar AUC but have overlapping ROC curves with different shapes
and different areas. For example, it might
be important to achieve very low false positive rates for a specific range of
true positive rates. But using only AUC
would not be able to necessarily detect those more subtle
performance differences.
