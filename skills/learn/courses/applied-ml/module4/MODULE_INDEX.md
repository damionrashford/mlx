- Recognize and avoid data leakage

## Key Concepts Covered

### Supervised Learning - Advanced
- Naive Bayes (Gaussian, Multinomial, Bernoulli)
- Random Forests (bagging + random features)
- Gradient Boosted Trees (sequential error correction)
- Feature Importance
- Neural Networks (MLPClassifier/MLPRegressor)
- Activation Functions (ReLU, sigmoid, tanh)
- Deep Learning Basics

### Data Leakage (CRITICAL)
- Target Leakage
- Train-Test Contamination
- Temporal Data Splits
- Pipeline Design

### Unsupervised Learning
- Dimensionality Reduction (PCA, t-SNE)
- Clustering (K-Means, DBSCAN, Agglomerative)
- Silhouette Score
- Within-Cluster Sum of Squares
