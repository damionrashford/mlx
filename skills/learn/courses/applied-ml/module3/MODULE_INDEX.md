# Module 3: Evaluation

## Overview
This module focuses on evaluation metrics for machine learning models. Learn why accuracy alone is inadequate and how to use appropriate metrics for different tasks.

## Learning Objectives
- Understand limitations of accuracy as an evaluation metric
- Learn and interpret various evaluation metrics
- Optimize ML algorithms using task-appropriate metrics
- Work with confusion matrices, ROC curves, and precision-recall curves
