# Module 2: Supervised Machine Learning - Part 1

## Overview
This module covers multiple supervised learning algorithms, focusing on linear models, support vector machines, and decision trees. Emphasis on model complexity, regularization, and cross-validation.

## Learning Objectives
- Understand how different supervised learning algorithms estimate parameters
- Apply linear models, SVMs, and decision trees in Python
- Recognize the connection between model complexity and generalization
- Apply regularization, feature scaling, and cross-validation techniques
