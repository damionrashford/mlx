# Module 1: Fundamentals of Machine Learning - Intro to SciKit Learn

## Overview
This module introduces basic machine learning concepts, tasks, and workflow using an example classification problem based on the K-nearest neighbors method, and implemented using the scikit-learn library.

## Learning Objectives
- Understand basic machine learning concepts and workflow
- Distinguish between different types of machine learning tasks
- Understand how K-nearest neighbors algorithm works
- Build and evaluate a basic k-nearest neighbors classifier using scikit-learn
